# Decisions (ADRs)

Architecture/product decision records. One file per decision, immutable once accepted — to change a
decision, write a new ADR that supersedes it.

- **Filename:** `NNNN-slug.md` (zero-padded, sequential).
- **Header:** `id, title, status, date, deciders`.
- **Status:** `proposed → accepted → (deprecated | superseded by ADR-NNNN)`.
